# avrlearn009-SPI
Piotr Makulec, 14JA2026
This repo is a part of my AVR self-learning journey.
The code was written in C for ATmega328p (Arduino).
The code controls 8 LEDs via the 74HC595 shift register.
Microcontroller communicates with the shift register using the hardware SPI protocol.
